# SRIT Canteen Pre-order Web App - Setup Guide

## First User Setup

**IMPORTANT**: The first user to register will automatically become an admin.

### Steps:
1. Go to the registration page
2. Register with any @srit.ac.in email address
3. You will be automatically logged in as an admin
4. As admin, you can:
   - Add other emails to the whitelist (Admin Panel → User Management)
   - Assign roles to other users
   - Manage menu items and sessions

### Adding Users to Whitelist:
1. Login as admin
2. Go to Admin Panel → User Management
3. Click "Add Email" button
4. Enter the @srit.ac.in email address
5. Users with whitelisted emails can now register

## Email Verification

Email verification has been disabled to avoid rate limit issues. Users are automatically logged in after successful registration.

## User Roles

- **Student/Staff**: Can browse menu and place orders
- **Admin**: Full access to all features including user management
- **Kitchen Staff**: Can manage orders and menu
- **Cashier**: Can manage orders and payments

## Default Menu Sessions

- Breakfast: 7:00 AM - 10:00 AM (60 min cutoff)
- Lunch: 12:00 PM - 3:00 PM (120 min cutoff)
- Snacks: 4:00 PM - 6:00 PM (60 min cutoff)
- Dinner: 7:00 PM - 10:00 PM (120 min cutoff)

## Sample Menu Items

The database includes sample menu items:
- Idli Sambar (Breakfast)
- Masala Dosa (Breakfast)
- Veg Thali (Lunch)
- Chicken Biryani (Lunch)
- Samosa (Snacks)
- Paneer Butter Masala (Dinner)

Admins can add, edit, or remove menu items from the Admin Panel.
